Class Trigger

Triggers a method of firing arbitrary code on collision. These are useful as 'buttons', 'switches', or to trigger effects in a game. By defualt triggers are invisible, and can only be seen with debug mode enabled on the Engine.

The height of the trigger

param

Callback to fire when trigger is activated

The number of times that this trigger should fire, by default it is 1, if -1 is supplied it will fire indefinitely
