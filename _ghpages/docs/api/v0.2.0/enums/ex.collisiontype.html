Active

Actors with the Active setting raise collision events and participate in collisions with other actors and will be push or moved by actors sharing the Active or Fixed setting.

Elastic

Elastic: CollisionType
  • Defined in src/engine/Entities.ts:328

Actors with the Elastic setting will behave the same as Active, except that they will "bounce" in the opposite direction given their velocity dx/dy. This is a naive implementation meant for prototyping, for a more robust elastic collision listen to the "collision" event and perform your custom logic.

Fixed

Fixed: CollisionType
  • Defined in src/engine/Entities.ts:339

Actors with the Fixed setting raise collision events and participate in collisions with other actors. Actors with the Fixed setting will not be pushed or moved by other actors sharing the Fixed or Actors. Think of Fixed actors as "immovable/onstoppable" objects. If two Fixed actors meet they will not be pushed or moved by each other, they will not interact except to throw collision events.
