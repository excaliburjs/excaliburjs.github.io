An [[Entity]] in is an empty container for [[Component|Components]]. On its own, an Entity has very little behavior.

Entities are useful for defining basic game objects with custom behavior.

If you want a preconfigured Entity, use an [[Actor]] (read more [here](/docs/actor)).
