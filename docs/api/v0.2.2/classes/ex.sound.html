Class Sound

The Sound object allows games built in Excalibur to load audio components, from soundtracks to sound effects. It is generally recommended to load sound resources when using Excalibur

class

Sound

extend

ILoadable

constructor
param

A list of audio sources (clip.wav, clip.mp3, clip.ogg) for this audio clip. This is done for browser compatibility.
