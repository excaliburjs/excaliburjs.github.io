Class Template

Excalibur's built in templating class, it is a loadable that will load and html fragment from a url. Excalibur templating is very basic only allowing bindings of the type data-text="this.obj.someprop", data-style="color:this.obj.color.toString()". Bindings allow all valid Javascript expressions.
