Class Actor

The most important primitive in Excalibur is an "Actor." Anything that can move on the screen, collide with another Actor, respond to events, or interact with the current scene, must be an actor. An Actor must be part of a {{#crossLink "Scene"}}{{/crossLink}} for it to be drawn to the screen.
