Class Label

Labels are the way to draw small amounts of text to the screen in Excalibur. They are actors and inherit all of the benifits and capabilities.

class

Label

extends

Actor

constructor
param

The text of the label

param

The x position of the label

param

The y position of the label

param

Use any valid css font string for the label's font. Default is "10px sans-serif".

param

Use an Excalibur sprite font for the label's font, if a SpriteFont is provided it will take precendence over a css font.
