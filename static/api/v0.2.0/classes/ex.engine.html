Class Engine

The 'Engine' is the main driver for a game. It is responsible for starting/stopping the game, maintaining state, transmitting events, loading resources, and managing the scene.

class

Engine

constructor
param

The width in pixels of the Excalibur game viewport

param

The height in pixels of the Excalibur game viewport

param

If this is not specified, then a new canvas will be created and inserted into the body.

param

If this is not specified, then it will fall back to fixed if a height and width are specified, else the display mode will be FullScreen.
